# Roadmap

## Phase 1 — platform and array baseline
- Freeze architecture and wiring
- Build first rigid 5-antenna array
- Verify coherent capture and channel health
- Record first baseline data sets

## Phase 2 — calibration and beamforming
- Implement calibration checks
- Build delay-and-sum reference pipeline
- Implement MVDR / adaptive interference-mitigation reference flow
- Compare host-only versus host+FPGA partitioning

## Phase 3 — FPGA-centered integration
- Define Pi ↔ PolarFire control path
- Move selected kernels to FPGA
- Validate data movement and latency
- Demonstrate meaningful FPGA contribution

## Phase 4 — validation and final demo
- Conducted / shielded anti-jam testing
- Runtime, coherence, suppression, and stability measurements
- Final report figures and presentation materials
- End-to-end demonstration
