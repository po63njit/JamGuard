# Course Docs

This folder stores course-provided material and assignment-specific reference documents.

Keep original files unchanged when possible. If you need annotated or working copies, place them in:
- `reports/` for deliverable drafts
- `project_management/` for planning notes derived from the course material

Current contents should stay source-of-truth for:
- project description and scope constraints
- grading and schedule requirements
- official proposal and reporting expectations
