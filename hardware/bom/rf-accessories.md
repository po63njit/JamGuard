# RF Accessories BOM

These items support controlled testing, protection, and debug. They are important, but they should not be confused with the core system path.

| Item | Qty | Role | Priority | Notes |
|---|---|---:|---|---|
| L1 filter | 1-2 | Reduce out-of-band energy during controlled tests | Useful support item | Record insertion loss and passband |
| Attenuators | Assorted | Protect front end and control signal levels | Useful support item | Keep multiple fixed values available |
| Terminators | Assorted | Properly terminate unused RF paths | Useful support item | Prefer quality 50 ohm parts |
| SMA adapters | Assorted | Connector adaptation during prototype integration | Useful support item | Avoid building the test setup around adapters long-term |
| Shielding materials / shield box access | 1 set | More repeatable interference experiments | Optional but beneficial | Helps bench validation more than baseline bring-up |

## Selection criteria
- Minimize adapter stacks.
- Record known insertion losses.
- Prefer parts that are reusable in both bench and field workflows.
