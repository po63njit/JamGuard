# Prototype 1 Geometry

## Objective
Define the first rigid five-antenna layout that is simple to build, mechanically stable, and good enough for coherent capture validation.

## Design intent
- Use a rigid plate or ground plane as the common mounting surface.
- Keep antenna placement repeatable and easy to measure.
- Route cables in a way that minimizes movement between runs.
- Prefer a geometry that can be rebuilt after disassembly without guessing dimensions.

## First-pass layout assumptions
- Five antennas total.
- One center element with four surrounding elements is the preferred starting geometry.
- Equal radial spacing from the center is easier to document than an irregular layout.
- Cable exits should be aligned so routing is consistent between channels.

## Dimensions to freeze next
- center-to-center antenna spacing
- overall plate dimensions
- antenna orientation reference
- mount height above ground plane
- cable bend and strain-relief path

## Acceptance criteria
- All five antenna positions are measurable and documented.
- Channel-to-channel cabling is labeled.
- The structure survives handling without position drift.
- Reassembly can reproduce the same geometry within a documented tolerance.
