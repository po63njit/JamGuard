# Hardware

This folder contains the physical implementation of the system: antenna-array design, BOM data, RF interconnect choices, power notes, and mechanical prototype details.

## Recommended structure
- `antenna_array/`: geometry drawings, spacing calculations, mount revisions, photos
- `bom/`: procurement-ready parts lists with quantities, costs, and alternatives

## High-priority missing content
- antenna spacing sketches with actual dimensions
- ground-plane and mount dimensions
- cable and adapter inventory
- power budget and connector notes
- photos of each prototype revision
- assembly notes that let you rebuild the array the same way twice
