# Host Software

Raspberry Pi 5 host-side scripts, setup notes, configuration, logging tools, and orchestration notes go here.

## Recommended subfolders
- `docs/`: bring-up notes, operator instructions, screenshots
- `configs/`: known-good config files and service definitions
- `scripts/`: repeatable utilities for setup, logging, and capture control
